Inventario - Productos Tecnológicos

Este proyecto consiste en un programa en Python que permite analizar el inventario de productos tecnológicos y determinar la cantidad de productos que deben ser solicitados según el stock actual y el stock mínimo requerido.

Funcionalidad
- Analiza 10 productos tecnológicos
- Calcula la cantidad a pedir
- Usa funciones y estructuras de control
